

* All intermediate files land in \compile\debug\projectname or \compile\release\projectname, the resulting
  programs, libraries and dlls land in \compile\debug\ or \compile\release

* The ML directory must be in the search path, 

* Scripts for windows cygwin 
	-	getheaders copies all header files to a directory
	-	fromdos	converts all c,cxx and h files to unix format.
	-	todos converts all dsw and dsp files to NT format so VC6 can read them
